# Memory Tree - 操作手册

## 1. 启动说明

本项目是一个基于手势交互的3D照片墙系统。

### 运行步骤
1. 确保已安装 Node.js。
2. 在项目根目录下运行 `npm install` 安装依赖。
3. 运行 `npm run dev` 启动开发服务器。
4. 打开浏览器访问显示的本地地址（通常是 http://localhost:5173）。
5. **允许摄像头权限**：系统需要访问摄像头以进行手势识别。

---

## 2. 操控指南

系统支持双手操作，通过 MediaPipe 进行手势识别。

### 🙌 双手控制 (NEW)
*   **合掌缩放**: 双手同时控制球体的缩放。
    *   **双手合掌**: 球体缩小到最小尺寸。
    *   **双手分开**: 球体放大到最大尺寸。
    *   需要双手同时被摄像头检测到才能生效。
    *   缩放更加平滑和直观。

### ✋ 右手控制
*   **移动 (Move)**: 控制球体的旋转。
    *   手掌在屏幕中移动，球体会跟随旋转（上下左右）。
    *   *注意*：当右手处于捏合状态时，旋转会暂停，以免误触。
*   **捏合 (Pinch)**: 选择/点击。
    *   将光标（绿色圆点）移动到照片上，捏合即可“选中”或“拖动”（当前交互为悬停放大）。
*   **双次捏合 (Double Pinch)**: 进入详情模式。
    *   光标悬停在某张照片上时，快速捏合两次，将该照片放大至全屏查看（Detail Mode）。

### 🖼️ 详情模式 (Detail Mode)
当进入单张照片详情页时：
*   **挥手 (Swipe)**: 切换照片。
    *   向左挥动：下一张。
    *   向右挥动：上一张。
*   **握拳 (Fist)**: 关闭详情。
    *   握紧拳头，保持约 0.5 秒，将关闭大图，返回球体预览模式。

### 🖥️ 界面功能
*   **右上角控制面板**:
    *   **Shape**: 暂时保留用于未来扩展（当前固定为球体布局）。
    *   **Immersive Camera**: 开启/关闭全屏摄像头背景。

---

## 3. 常见问题
*   **摄像头不清晰**：请确保光线充足，且浏览器未对摄像头进行缩放或限制。系统默认尝试使用 1280x720 高清分辨率。
*   **手势无反应**：请确保手部完全在摄像头画面内，且背景不要过于杂乱。
*   **加载失败**：如遇“Vision Load Failed”，通常是网络问题导致无法下载 MediaPipe 模型文件，请检查网络连接。
